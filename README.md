## Develop

```bash

npm run build    # production build -> dist/
```

## Media

Drop background videos / images into `public/media/` using the exact names in `public/media/README.md`. They appear automatically; until then an animated gradient fallback shows.

## SEO / AEO

Self-canonical to `https://backend.mkazi.live`, keyword-optimized titles/meta, JSON-LD (`Person` + `FAQPage`), `sitemap.xml` and `robots.txt`. Discipline landing pages live under [/backend-developer](https://backend.mkazi.live/backend-developer) and `/portfolio`.

## Deploy

Deployed on **Vercel** (`vercel.json`) — pushes to `main` deploy automatically. Domain: `backend.mkazi.live`.
